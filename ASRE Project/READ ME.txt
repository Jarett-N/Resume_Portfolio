This is my project for my capstone project at UWEC, we created an app to interact with NAO v5/v6 robots. We had a webpage locally hosted on a laptop, in which we could have the app displayed and controlled by anyone, we had a D-Pad for movement, a space for Text To Speech, and self creating buttons for animations for the robot to do, which were created on files of the animations being loaded onto the app. All of this was stored locally in a JSON file.
